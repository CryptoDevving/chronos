# Chronos

Proof-of-concept [TWAMM](https://www.paradigm.xyz/2021/07/twamm/).

## Install dependencies

```sh
npm i
```

## Run local dev server

```sh
npm run dev
```

## Build for production

```sh
npm run build
```

## Preview production build

```sh
npm run preview
```
